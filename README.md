# PyPropagate

A paraxial wave propagation framework for python 2.7 . 

## Installation
    
    pip install pypropagate

## Dependencies

- Python 2.7
- gcc >= 5.0 or compatible
- boost python
- xraylib

Python dependencies are installed automatically via pip.
